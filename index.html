Absensi Ujian Digital

A simple yet functional application built with native PHP. This project focuses on speed and ease of use for exam supervisors to record participant attendance.

Workflow:

  • Supervisor logs into the system.
  • Selects exam session and room.
  • Displays list of exam participants in that room.
  • Marks attendance status (Present, Absent, Late) for each student.
  • Saves data to database in real-time.
PHP MySQL JavaScript AJAX HTML/CSS

VolunTeam Platform

A centralized volunteer management system engineered with Laravel 10. This platform solves the complexity of recruiting, tracking, and managing volunteers for large-scale events.

Built with Clean Architecture principles to ensure maintainability and scalability for future development.

Key Technical Features:

  • Secure Authentication: Multi-guard login system for Admins, Organizers, and Volunteers.
  • Advanced CRUD: Efficient data handling for Event creation and Volunteer application tracking.
  • MVC Architecture: Separation of logic (Controller) and interface (View) for cleaner code.
  • Responsive UI: Built with Bootstrap 5 to ensure accessibility across devices.
